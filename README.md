# Scheme
Минимальный интерпретатор языка [Scheme](https://www.scheme.org/) на C++, написанный в рамках курса [cpp-advanced-hse](https://gitlab.com/hse-cpp/cpp-advanced-hse/-/tree/main/tasks/scheme?ref_type=heads).

## Сборка и запуск

```sh
just run-release
```

# Возможности

Пока реализованы только самые простые конструкции:
* работа с целыми числами (пока только int64)
* булевы значения (#t и #f)
* идентификаторы (они же symbols) для именования переменных, лямбда-выражений, встроенных функций
* списки и пары
* переменные и лямбда-выражения
* некоторые специальные формы и функции для работы с целыми числами, булевыми значениями, списками
* сборка мусора

## Структура

Сейчас единственный вариант использования - REPL-консоль. Сам интерпретатор разбит на несколько частей:

- `tokenizer/` читает поток символов и превращает его в токены: скобки, точку, quote, числа и
  символы.
- `parser/` принимает поток токенов и строит по ним AST (Абстрактное синтаксическое дерево).
- `runtime/` содержит объектную модель языка, окружения, встроенные функции, специальные формы и
  сборщик мусора.
- `interpreter/` связывает парсер и рантайм: читает одно выражение, вычисляет его в глобальном
  окружении и сериализует результат.
- `repl/` простой REPL.

## Примеры
Факториал:
```scheme
(define (fact n)
  (if (= n 0)
      1
      (* n (fact (- n 1)))))

(fact 5)
```

Числа Фибоначчи:
```scheme
(define (fib n)
  (if (< n 2)
      n
      (+ (fib (- n 1)) (fib (- n 2)))))

(fib 10)
```

Быстрое возведение в степень:
```scheme
(define (even? n)
  (= 0 (- n (* (/ n 2) 2))))

(define (pow-fast x n)
  (if (= n 0)
      1
      (if (even? n)
          (pow-fast (* x x) (/ n 2))
          (* x (pow-fast x (- n 1))))))

(pow-fast 3 20)
```

Проверка наличия элемента в списке:
```scheme
(define (contains? xs value)
  (if (null? xs)
      #f
      (if (= (car xs) value)
          #t
          (contains? (cdr xs) value))))

(contains? '(4 8 15 16 23 42) 15)

(contains? '(4 8 15 16 23 42) 7)
```
